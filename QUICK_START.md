# Quick Start Guide

## 🚀 Get Started in 5 Minutes

### Step 1: Install the Extension

1. Go to `chrome://extensions/`
2. Enable "Developer mode" (toggle, top right)
3. Click "Load unpacked"
4. Select the `parentalControl` folder
5. Click "Select Folder"

You should see the extension appear in your list and an icon in the Chrome toolbar.

### Step 2: Initial Setup (First Time Only)

The settings page will automatically open. You'll see:

1. **Create a Master Password**
   - Go to "Security" section
   - Enter a password (min 4 characters)
   - Click "Update Password"
   - This protects access to settings

2. **Add Domains to Block**
   - Go to "Block List" section
   - Type a domain (e.g., `facebook.com`)
   - Click "Add Domain"
   - Repeat for other domains

3. **Create a Schedule (Optional)**
   - Go to "Time-Based Schedules"
   - Select "Restricted" or "Allowed" mode
   - Set start and end times
   - Choose days of the week
   - Click "Add Schedule"

4. **Save Your Settings**
   - Click "Save All Settings" at the bottom
   - You should see a success message

### Step 3: Test It Works

1. Try visiting a domain you blocked
2. You should see a "Access Blocked" page
3. Click the extension icon in Chrome to see status

## 📋 Common Tasks

### Block a Website
1. Click extension icon → "Settings"
2. Enter password
3. In "Block List" section, add the domain
4. Click "Save All Settings"

### Allow During School Hours
1. Go to Settings → "Time-Based Schedules"
2. Set mode to "Allowed"
3. Start time: 09:00, End time: 16:00
4. Select Mon-Fri
5. Click "Add Schedule"
6. Save Settings

### Restrict During Bedtime
1. Go to Settings → "Time-Based Schedules"
2. Set mode to "Restricted"
3. Start time: 21:00, End time: 07:00
4. Select all days
5. In "Always Allowed List", add homework sites like `google.com`
6. Click "Add Schedule"
7. Save Settings

### Check What Was Blocked
1. Click extension icon → "View Logs"
2. See all blocked attempts
3. Can export as CSV

## 🔐 Security Tips

- Use a strong password (8+ characters)
- Don't use the same password as other accounts
- Check logs regularly
- Combine with device-level parental controls for better protection

## ❓ Need Help?

- **Can't remember password?** You must uninstall and reinstall (starts fresh)
- **Domain still accessible?** Check the exact domain name in URL bar
- **Schedule not working?** Verify browser time is correct
- **Forgot to add a domain?** Add it anytime and click "Save All Settings"

See `INSTALLATION.md` for more detailed information.
